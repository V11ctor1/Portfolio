# Портфолио

### _Структура приложения_
```
|-- static ~ папка статических файлов
|   |--css/main.css ~ стили
|   |-- images/ ~ изображения
|   |--js/main.js ~ шапка сайта js
|-- templates ~ папка шаблонов html
|   |--base.html ~ Содержит шапку и подвал страницы
|   |--index.html ~ Главная страница
|   |--works_base.html ~ Содержит оформление работ
|   |--works.html ~ заглушка станицы работ
|   |--weather.html ~ погода после ввода города
|   |--weather_form.html ~ форма ввода города погоды
|   |--well.html ~ ошибка 404
|-- main.py ~ исполняемый файл

```
